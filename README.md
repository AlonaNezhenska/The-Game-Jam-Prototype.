# 🎮 Office Game – Financial Literacy Game

A React-based interactive game developed as part of the **Lloyds Banking Group × Ada Scotland Game Jam 2025**.  
The project focuses on making financial literacy more engaging and accessible through gameplay.

---

## 🏆 Achievement

This project was part of the **winning submission** at the 2025 Lloyds Banking Group × Ada Scotland Game Jam.  
The game was later deployed on the **Scottish Widows website**.
